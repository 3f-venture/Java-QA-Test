# QA Test Task (Web + API, Java + Selenide + RestAssured)

## Описание

Это тестовое задание на позицию QA Engineer.

В проекте уже есть каркас для:
- UI-тестов (Selenide)
- API-тестов (RestAssured)

Ваша задача — **дописать реализацию** и **написать тесты** согласно разделу “Что нужно сделать”.

---

## Что нужно сделать

### 1. UI (Selenide)

Приложение: используйте любой публичный демо-сайт (например, https://the-internet.herokuapp.com).

1. Реализуйте методы в `LoginPage` (Page Object).
2. Напишите 2–3 автотеста в `LoginTests`:
   - успешный логин;
   - неуспешный логин с неверными данными;
   - (опционально) проверка сообщения об ошибке.

### 2. API (RestAssured)

API: используйте публичное API, например: https://reqres.in.

1. Реализуйте базовую конфигурацию в `BaseApiTest`.
2. Напишите 3–5 тестов в `UserApiTests`:
   - получение списка пользователей;
   - успешное создание пользователя;
   - получение конкретного пользователя;
   - негативный сценарий (например, пользователь не найден);
   - (опционально) удаление пользователя.
